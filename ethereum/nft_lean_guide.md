# NFT 学习入门指南

## 什么是 NFT

非同质代币(NFT)是一种具有唯一性识别的代币。 non-fungible这个词意味着你不能用一个NFT来代替另一个NFT。 每一个NFT都是独一无二的。 相比比特币或ERC-20等任何可互换的代币，你收到的一个币（coin）/代币（token）并不重要，它们都具有相同的价值。

## ERC721 协议

不可替代代币的标准接口，也称为契约。
